# SealAI Keyword Strategy Seed

This is a starting keyword map until GSC query data becomes statistically useful.

Status: seed strategy, not volume-validated.

## Decision Rule

Prioritize keywords where SealAI can answer a high-intent technical question better than a catalog page.

Good fits:

- unclear sealing problem
- material compatibility question
- operating condition / medium / temperature / pressure context
- RFQ preparation
- manufacturer-review-ready inquiry

Weak fits:

- pure shop SKU searches
- price-only searches
- final approval/certification searches without evidence

## GSC Quick-Win Interpretation

The weekly report should win over this seed list once query rows exist.

Manual priority:

1. Impressions >= 100 in 28 final days
2. Average position 5-20
3. CTR below expected threshold
4. Page intent actually matches the query
5. No final engineering approval claim required

## Source Notes

Google documents GSC performance metrics as clicks, impressions, CTR, and average position, and recommends filtering/grouping by query/page for analysis.

Market SERP spot checks show the German sealing space is heavily catalog/standard-page oriented around DIN 3760, Radialwellendichtring, FKM, NBR, PTFE, and Wellendichtring terminology. SealAI should not compete as a pure catalog first; it should compete on problem clarification and manufacturer-ready inquiry preparation.

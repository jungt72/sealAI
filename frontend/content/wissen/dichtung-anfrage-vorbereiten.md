# Dichtung Anfrage vorbereiten: technische Angaben für Hersteller strukturieren

Eine gute Dichtungsanfrage beschreibt nicht nur ein Bauteil, sondern den Betriebsfall. Hersteller können nur dann sinnvoll prüfen, wenn Medium, Temperatur, Druck, Bewegung, Bauraum und Anforderungen klar genug sind.

sealingAI hilft, bekannte Daten und offene Punkte sauber zu trennen. Das Ziel ist eine herstellerprüfbare Anfragebasis.

## Mindestangaben für eine technische Dichtungsanfrage

| Bereich | Beispiele |
|---|---|
| Anwendung | Pumpe, Welle, Flansch, Ventil, Gehäuse, Sonderbauteil |
| Medium | Bezeichnung, Konzentration, Additive, Verschmutzung |
| Temperatur | Dauerbetrieb, Spitzen, Reinigungszyklen |
| Druck | Betriebsdruck, Spitzen, Vakuum, Druckwechsel |
| Bewegung | statisch, rotierend, oszillierend, Hub |
| Bauraum | Maße, Toleranzen, Normbezug, Zeichnung |
| Ziel | Ersatz, Neuauslegung, Optimierung oder Schadensanalyse |

## Warum offene Punkte sichtbar bleiben sollten

Unvollständige Angaben sind normal. Kritisch wird es erst, wenn Annahmen als Fakten behandelt werden. Eine professionelle Anfrage zeigt, welche Daten gesichert sind und welche durch Hersteller oder Betreiber noch geprüft werden müssen.
